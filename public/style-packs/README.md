# Style Pack Guide

Untuk setiap style, isi folder dengan **nama file ini persis**:

1. `action-1.(png|jpg|jpeg|webp)`
2. `action-2.(png|jpg|jpeg|webp)`
3. `action-3.(png|jpg|jpeg|webp)`
4. `closeup-1.(png|jpg|jpeg|webp)`
5. `environment-1.(png|jpg|jpeg|webp)`
6. `style_profile.json`

Spesifikasi:
- Sisi panjang minimal 1024 px
- Tanpa watermark / teks overlay
- Framing karakter jelas

Contoh struktur:

```txt
public/style-packs/naruto/
  action-1.png
  action-2.png
  action-3.png
  closeup-1.jpg
  environment-1.webp
  style_profile.json
```
